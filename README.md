# Dr. Cinema  - Group 4
Dr. Cinema is a React Native application that displays movies currently playing in cinemas.

## Get started
1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## This project was fully tested on iOS using Expo Go.
You can scan the QR code using the Expo Go app on an iPhone, or press i in the terminal to open the project in the iOS simulator (macOS required).
